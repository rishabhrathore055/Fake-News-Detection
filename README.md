# Fake News Detection
In this project, i used sklearn libraries from Python to predict the false news articles with using natural language processing techniques and machine learning models.
## Dependencies
* Pandas
* Numpy
* Sklearn
   * LinearRegression
   * Ridge
   * RidgeCV
   * metrics
* Nltk
   * PorterStemmer
   * TfidfVectorizer
